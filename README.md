# ECE452 (MunchBox)

## Firebase

After downloading the `google-services.json` add it to your **(app-level)** root directory. This file is basically our API key for using Firebase services.  
**Example:**  
>     Munchbox/...
> 
>        └── App/...
> 
>           └── Src/...
> 
>           └── google-services.json <--(put it here)
>
>           └── ...
